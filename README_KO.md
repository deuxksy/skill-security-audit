# security-audit

OpenClaw용 크로스 플랫폼 보안 취약점 및 업데이트 점검 도구.

## 설치

```bash
git clone git@github.com:deuxksy/skill-security-audit.git ~/.openclaw/skills/security-audit
```

OpenClaw가 `~/.openclaw/skills/`에서 자동으로 스킬을 로드합니다. 재시작 불필요.

⚠️ **심볼릭 링크 사용 금지** — OpenClaw는 보안상 설정된 루트 외부를 가리키는 경로를 로드하지 않습니다. 반드시 직접 clone 하세요.
